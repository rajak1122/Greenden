# Greenden 🌿

Greenden is a modern and responsive plant shop website built using HTML, Tailwind CSS and JavaScript.

## 🚀 Live Demo

https://rajak1122.github.io/Greenden/index.html

## ✨ Features

- Responsive design
- Modern UI
- Tailwind CSS styling
- Product showcase section
- Clean layout
- Mobile-friendly navigation
- Search functionality on product page
- On mobile view the menu icon becomes functional to nagivate to other pages

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
